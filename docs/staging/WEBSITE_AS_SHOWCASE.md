THE VISION (Rich's words, structured). The website is the showcase to the outside world of how amazing this idea is. A stranger should feel it within one minute. Each tab has a job:

1. SIM TAB -- BRING THE WORLD TO LIFE: the physics, event frequencies, behaviours and interrelationships as living visuals from the latest run: weather->demand->price signal chains; event-rate panels (life events, bill shocks, complaints, switches per year); journey-stage flows (how many customers sit in content/irritated/in-market right now, transitions over time); correlation panels (income stress vs payment delay, price vs in-market entry, satisfaction vs reviews). The synthetic world as a nature documentary, not a table dump.

2. SUPPLIER TAB -- OBSERVATION -> ACTION -> EFFECT: what the company observed, what it decided (event ledger), and what VALUE the action created. THE HEADLINE METRIC OF THE WHOLE SITE: enterprise value created VERSUS DOING THE SAME AS LAST TIME. Build the frozen-policy baseline: replay the same decade with last-generation policies (pre-decision-loop, pre-deferral-pricing, naive retention) vs current policies -> delta-EV = THE VALUE OF LEARNING. "Did decades of operation make it better" answered with a number and a learning curve, run over run. This requires policy-snapshot + replay mechanics -- design that piece first (Tier 3 design note; implementation Tier 2 after).

3. PROJECT TAB -- THE ACCUMULATION: the learning ledger. Discoveries, policy generations, delta-EV per generation -- Digital Darwinism made visible as a compounding curve, plus the SaaS coverage map and the retrospectives. This is the investor room.

4. CUSTOMER PORTAL -- MICRO MEETS MACRO: for a single customer, show the macro actions landing on their life -- the hedge that set their tariff, the decision-loop offer at their renewal, the dunning path on their arrears. Plus the CASE-STUDY RECOMMENDER: the site auto-curates "interesting customers" -- most eventful journeys, largest company-vs-SIM divergences, defer-then-churn sequences (C1/C5/C6), the C7 new-baby case -- each linking straight to that customer's timeline. Let the project recommend where to look.

QUALITY BAR: professional corporate product on site/ (consistent design system, restrained, data-dense); everything generated from the canonical run artifact (consistency gate applies); freshness stamps; works as a first impression for an investor who gives it 60 seconds.
