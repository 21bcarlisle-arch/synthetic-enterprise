# COLD_EYES_PROTOCOL — manufacture the outside vantage (Tier 2, Lane H)

**Staged:** 2026-07-11 by advisor, from the director's insight: the advisor
catching the thesis-chart defects was not superior capability — same model —
it was VANTAGE (cold context, audience-primed, artefact-only). So
institutionalise that vantage as a repeatable role. Non-interrupting; next
boundary. This upgrades the existing Expert-Hour simulation and the
fresh-context evaluator — same machinery, stricter blindfold.

## The protocol (for every Expert-Hour walk and every Door/page close)
1. **Fresh instance, hard blindfold:** the reviewer sees ONLY the deployed
   rendered artefact (URL/screenshot), never the build context, diffs, or
   intentions. Epistemic wall applied to review: the grader must not see
   inside the build.
2. **Priors before pixels:** before viewing, the reviewer is asked to STATE
   expected sane ranges for the figure classes it is about to see (e.g.
   "incumbent opex per household: £X-Y"). Then it looks. Any displayed figure
   outside its own stated priors = finding. This forces the plausibility
   knowledge that exists in the weights to actually fire.
3. **Persona priming:** run as the named veteran (energy CEO / CTO / VC /
   payments ops), with the persona's first question: "what would you doubt
   first on this page?" Doubts are findings even when the underlying number
   is internally correct — reads-as matters.
4. **Same-page reconciliation:** the reviewer explicitly cross-checks every
   figure appearing more than once or in tension on one page (the 11-vs-13
   class), independent of the data-layer consistency gate.
5. Findings land in the adjudication ledger like any other; builder context
   is only consulted AFTER findings are filed.

## Plausibility anchors (standing, complements the protocol)
Every headline figure class on public surfaces gets an independently-sourced
sane-range anchor in the invariants library (independence rule applied to
PRESENTATION: the anchor must not derive from our own pipeline). Out-of-range
display = sanity-daemon flag, even when internally correct — "correct but
implausible-looking" is a finding class, resolved by either fixing the basis
or explaining the divergence ON the surface.
